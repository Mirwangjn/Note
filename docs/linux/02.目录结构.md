- `/`，根目录是最顶级的目录了
- Linux只有一个顶级目录：`/`
- 路径描述的层次关系同样适用`/`来表示
- `/home/itheima/a.txt`，表示根目录下的home文件夹内有itheima文件夹，内有`a.txt`
